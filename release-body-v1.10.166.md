**Supervertaler Workbench v1.10.166** — small follow-up to the v1.10.165 Prompt Manager restructure: a visible contextual help button.

## What's new

- **Contextual "?" help link in the Prompt Manager.** A small circular `?` button now sits on the same bottom row as **👁 Preview Combined**. One click opens the Prompt Manager help page (`help.supervertaler.com/workbench/ai-translation/prompt-library/`) in your default browser — the same destination F1 already reached via the existing context-help tagging, but now visibly discoverable next to the controls. Uses the shared `HelpButton` widget so the styling matches every other in-app `?` affordance.

If you're already on v1.10.165, this is a pure UI tweak — no other behavioural changes.

## Install

Download the Windows ZIP attached below, extract, and run `Supervertaler.exe`. Double-click `Add Supervertaler to Start Menu.cmd` (also in the ZIP) if you'd like a Start Menu shortcut.
