Features

Professional terminology and AI tools inside Trados Studio

🤖

AI Translation

A complete AI translation system with a project-aware chat assistant, batch translation, and a prompt library.

  • AI Assistant – dockable chat panel with access to your segment, terms, and TM matches
  • Batch Translate – translate selected segments, untranslated, or the entire document
  • Clipboard Mode – use any web-based AI (ChatGPT, Claude, Gemini) without an API key
  • QuickLauncher (Ctrl+Q) – one-click access to custom prompts
  • AutoPrompt – AI generates a domain-specific prompt from your document
  • OpenAI, Anthropic, Google, OpenRouter, and Ollama providers
  • Incognito Mode – anonymises project data for screen sharing and demos
🔎

TermLens

Inline terminology display with colour-coded chips for every matched term, right inside the Trados editor.

  • Colour-coded chips: blue, pink (project), yellow (non-translatable), green (MultiTerm)
  • Click or Alt+1–9 to insert terms into target
  • MultiTerm .sdltb support – reads termbases attached to your project
  • Quick-add shortcuts: Alt+Down, Alt+Up, Ctrl+Alt+T
  • TermPicker (Ctrl+Shift+G) for segments with many matches
  • Non-translatable term management (Ctrl+Alt+N)
🔍

SuperSearch

Cross-file search and find & replace across all SDLXLIFF files in a project – not just the file you have open.

  • Search source, target, or both with case-sensitive and regex support
  • Results grid with match highlighting and confirmation status
  • Preview pane – single-click a result to see the full segment text
  • Click-to-navigate – double-click to jump to the segment in the editor
  • Find & Replace in target text with Replace All safety confirmation
  • Alt+S to open; auto-fills from editor selection
🧠

SuperMemory

A self-organising translation knowledge base that the AI consults automatically when translating.

  • Stores client profiles, terminology decisions, domain conventions, and style preferences
  • Human-readable Markdown files – edit in any text editor or Obsidian
  • Auto-detection – matches client profiles and domain articles to your Trados project
  • Quick-add terms and corrections with Ctrl+Alt+M
  • Studio Tools – ask natural-language questions about your projects, TMs, and termbases
